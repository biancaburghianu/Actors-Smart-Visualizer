1. Abstract

Actors Smart Visualizer este un instrument Web de vizualizare flexibila a datelor referitoare la nominalizarile actorilor la Screen Actors Guid (SAG) Awards.

2. Introducere

Un utilizator poate vedea statisticile in functie de an, numele show-ului, al filmului sau al actorului. O alta functionalitate pe care o permite acest instrument Web este aceea de a descarca statisticile in diferite formate: CSV, WebP si SVG. Utilizatorul poate de asemenea sa citeasca articole referitoare la productii sau actori si sa vada nominalizarile in functie de categorii.

Scopul aplicatiei:

  • se incadreaza in domeniul family,free-time
  • este facuta pentru orice iubitor de filme, seriale
  • aduce o modalitate simpla si placuta de a ramane la curent cu nominalizarile la Screen Actors Guid (SAG) Awards si vietile actorilor
  • permite pastrarea statisticilor in formatul preferat

3. Structura

Site-ul este de tipul single-page aplication, are un API la care se fac request-uri si un server care retine continutul static al aplicatiei, avand de asemenea o baza de date in care vom salva informatiile legate de un utilizator, actorii si productiile preferate. Informatiile oferite unui utilizator legat de actori, nominalizari si productii vor fi de fapt raspunsuri ale request-urilor facute la API-ul The Movie Database (TMDb), iar statisticle vor fi preluate din Dataset-ul Screen Actors Guild Awards, 1994 - 2020.

3.1 Tehnologii utilizate

  • HTML
  • CSS
  • Node.js

3.2 Paginile site-lui

Fiecare pagina a site-lui se bazeaza pe cate un schelet html, toate paginile de html se gasesc in src/views/HTML. Stilizarile paginilor se gasesc in src/static/CSS, pentru fiecare schelet de HTML, numele CSS-urilor corespunde cu numele HTML-urilor, fiind diferita doar extensia fisierului. In plus, in src/static/JS avem implementate script-uri de JavaScript in care se gasesc functiile necesare pentru schetele de HTML, conventia de denumire este aceeasi ca si la HTML-uri si CSS-uri.

3.3 Pagina de login si register

Pagina de login asigura o modalitate utilizatorului de a se loga printr-un formular pentru ca acesta sa poata beneficia de serviciile instrumentului Web. In cazul in care utilizatorul nu si-a creeat inca un cont, poate accesa aceasta functionalitate din pagina formularul de Login, ii va fi afisat un formular aferent inregistrarii pe platforma noastra.

3.4 Pagina acasa

Aceasta pagina pune la dispozitie utilizatorului o lista cu productiile sale preferate, detalii despre noile productii, articole, statistici si nominalizari.

3.5 Pagina de statistici

In aceasta pagina utilizatorul poate vizualiza statisticile nominzalizarilor la Screen Actors Guid (SAG) Awards, le poate filtra in functie de denumire. Aceste statistici pot fi pe urma exportate si descarcate in urmatoarele formate: WebP, SVG si CSV.

3.8 Pagina de nominalizari

Aceasta pagina a fost creata pentru vizualizarea nominalizarilor la Screen Actors Guid (SAG) Awards. Utilizatorul poate filtra rezultatele in functie de categoria nominalizarilor: Movies, Stunts, Series.

3.9 Pagina cu informatii suplimentare

Daca un utilizator isi doreste sa afle mai multe informatii legate de actorii, filmele, sau serialele nominalizate, va fi redirectionat catre aceasta pagina, unde poate vizualiza mai multe detalii in functie de preferintele sale.

4. Caracteristicile site-ului(Goals)

Site-ul va folosi cel mai probabil un model arhitectural MVC(model-view-controller), pentru a fi mai usor de impartit pe categorii de lucru, facilitand astfel si munca in echipa. Vom folosi un controller pentru a parcurge diferite rute catre continutul pe care dorim sa il afisam pe pagina, deoarece vom crea un site de forma "single-page application". Vom retine date importante intr-o baza de date relationala, MySQL. Vom putea exporta statistici in format WebP, SVG si CSV.

5. Constrangeri

Nu exista constrangeri de utilizare, deoarece site-ul e conceput pentru orice tip de ecran, functionand si pe modul orizontal al unui telefon, tablete, etc partea de front-end fiind full responsive.

6. Functionalitatea site-ului

Site-ul este construit in asa fel incat utilizatorii sa fie atrasi de interfata placuta si usor de utilizat pentru o interactiune plina de beneficii. Utilizatori pot fi considerati doar cei logati, pentru a putea beneficia de servicii. Site-ul pune la dispozitie utilizatorului diverse functii precum crearea unui cont, logarea, folosirea contului pentru a vizualiza statisticile generate, a le descarca. De asemenea, utilizatorul poate vizualiza detalii suplimentare referitoare la productiile sale favorite, inclusiv actorii sai favoriti. Prin intermediul site-ului nostru, utilizatorul poate vizualiza si nominalizarile per categorie, poate sa isi salveze actorii si productiile favorite, astfel interactiunea sa cu site-ul nostru fiind strict legata de preferintele sale in materie de filme si seriale.
